# YouTube 免费节点提取工具

## 使用方法

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行程序

```bash
python main.py
```

## 程序流程

```
1. 获取 YouTube 博主 "ZYFXS" 的最新视频列表
       ↓
2. 查找标题包含 "免费节点" 的视频
       ↓
3. 从视频描述中提取 "下载地址" 后的 PrivateBin 链接
       ↓
4. 下载视频字幕
       ↓
5. 从字幕中提取 "密码" 后面的 4 位数字
       ↓
6. 使用密码解密 PrivateBin 链接
       ↓
7. 从解密内容中提取 V2Ray 订阅链接
```
